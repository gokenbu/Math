### 1. Latex基础

#### 空白

LATEX 将空格和制表符等空白字符视为相同的空白距离（space）。多  
个连续的空白字符等同为一个空白字符。在LATEX 文件中，每行开始的空  
白字符将被忽略，而单个的回车符被视为一空格。

#### 特殊字符

\# $ % ^ & \_ { } ~

使用这些字符时，可在前面加上反斜线

```
\# \$ \% \^{} \& \_ \{ \} \~{}
```

使用'\'则需要输入

```
$$\backslash$$
```

### 命令
